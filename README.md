# BacktoBack Finance contract 

This repo contains the contracts of BacktoBack Finance. 

## Functionality

- Vault
- Swap
- Redeem & Mint

## Installing

```shell
    npm install
```

## Available script
*Compile*
```shell
    npm run compile
```
*test*
```shell
    npm run test
```

*lint*
```shell
    npm run lint
```
